Using ``say_my_name``
------------------------

First import
~~~~~~~~~~~
First import ``say_my_name`` from the ``3-say_my_name.py`` module. Since the file name contains a numeral, you can't use the simple ``from 3-say_my_name.py import say_my_name``. You can instead use the following syntax:

    >>> say_my_name = __import__('3-say_my_name').say_my_name

Usage:
The function takes in two strings  first_name & last_name.
    say_my_name('Emmy', 'Chebet')
    My name is Emmy Chebet

  only one name (space before newline):
    >>> say_my_name('Emmy')
    My name is Emmy


  Exceptions
  If first name string argument is not a string, a TypeError of "first_name must be a string" will arise.
  Equally "last_name must be a string" for the last name argument.

last name is not a string:
